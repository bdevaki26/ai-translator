# AI Translator

An AI-powered language translator built with Streamlit and Groq API. Users can translate text between languages and choose the tone of the translation.

## How to Run
1. pip install -r requirements.txt
2. streamlit run app.py

## Tech Stack
Python, Streamlit, Groq API

## Features
- Translate text between multiple languages
- Tone control: Formal, Casual, Professional
- Clean and simple UI with Streamlit
